# ModYu HT4 Aftercare Assistant

An AI-powered chatbot demo built for **ModYu**, a UK-based startup developing a protocol-led scalp care pathway for hair transplant clinics.

🔗 **Live Site:** https://dalpoi.github.io/Modyu-demo

---

## Overview

This tool is designed to support patients recovering from hair transplant procedures and clinic staff using the HT4 protocol. Powered by Claude AI, it provides real-time, conversational guidance on post-procedure care — reducing repetitive patient queries and improving consistency of aftercare.

---

## HT4 Protocol Phases

| Phase | Stage | Description |
|-------|-------|-------------|
| Phase 1 | Pre-procedure | Scalp preparation |
| Phase 2 | Immediate post-procedure | Sterile wound care (CE-marked medical device) |
| Phase 3 | Early recovery | Gentle cleansing and soothing |
| Phase 4 | Ongoing maintenance | Long-term scalp health |

---

## Features

- Real-time AI responses powered by Claude (claude-sonnet-4-6)
- FAQ quick-access buttons for common patient questions
- Full conversation memory within a session
- Suitable for both patients and clinic staff
- Lightweight — single HTML file, no dependencies

---

## How to Use

1. Visit the [live site](https://dalpoi.github.io/Modyu-demo)
2. Enter your Anthropic API key in the input field
3. Click **Save Key**
4. Ask any question about hair transplant recovery or the HT4 protocol

---

## Built With

- Vanilla HTML / CSS / JavaScript
- [Anthropic Claude API](https://www.anthropic.com) for AI responses
